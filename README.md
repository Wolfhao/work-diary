# Work Diary 工作日记

> 专为宠物博主打造的商务接单管理工具

一个轻量、专注的**商务接单全周期管理平台**，涵盖商单录入/追踪、双账务（垫付 + 收入）看板、状态流转管理，帮助宠物博主从繁杂的接单事务中解放出来，专注内容创作。

## 核心功能

| 功能 | 说明 |
|------|------|
| 🔐 微信一键登录 | 通过 `wx.login()` code 换取系统 Token，自动完成注册 |
| 📋 商单全生命周期管理 | 待开工 → 制作中 → 待结款 → 已完成，支持截图附件上传 |
| 💰 双账务追踪 | 分别追踪垫付（是否已收回）与收入（是否已到账）状态 |
| 📊 数据看板 | 聚合统计总垫付、待收回垫付、总收入、已到账收入等关键指标 |
| 🗂 多维筛选 | 按商单状态、发布平台（小红书/抖音/B站等）进行筛选 |
| 🖼 文件上传 | 支持本地存储/阿里云 OSS/MinIO 灵活切换，可配置热更新 |
